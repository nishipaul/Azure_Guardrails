About this Interface

This interface allows you to test PII (Personally Identifiable Information) detection and LLM query processing. Enter a username and query to see how the system handles your request.

This will be used to fetch the appropriate prompt template from Langfuse
This specifies which version of the prompt template to use from Langfuse
Example:
Username: allinsale
Label: latest
Query: Where can I find the product in the store of Kolkata?
